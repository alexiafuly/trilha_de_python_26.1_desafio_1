# Trilha de Python 26.1 - Desafio 1: Sistema de Auxílio ao Orçamento de Viagens.
Trilha de Python da for_code onde aprendemos sobre python em nível intermediário.

## Funcionamento do programa:
O programa funciona como uma calculadora de viagens, na qual o usuário inserfornece algumas informações básicas sobre o orçamento e os gastos e o programa processa esses dados de entrada para calcular os custos e analisar a viabilidade da viagem.

## Instruções de uso
1) O usuário deve inserir todos os dados pedidos pelo programa: orçamento, destino da viagem, custo da passagem, custo diário de hospedagem e duração da viagem. É importante que todos os números inseridos sejam positivos, caso contrário, o programa dá erro.
2) Após a inserção dos dados, o programa calcula e exibe todos as informações necessárias para que a viagem seja possível: o custo total da hospedagem em reais, o custo total da viagem em reais, se o orçamento é possível ou não, se a viagem é viável ou não, e a quantidade de dinheiro que sobrou/faltou no orçamento.
3) Com essas informações, é só se planejar e pegar o próximo vôo. Boa viagem! :airplane:

## Respostas às perguntas teóricas
1) Qual a diferença entre o comando git add . e git commit -m "mensagem"?

R: git add . serve para adicionar um arquivo no repositório

2) Por que é necessário realizar o casting (conversão de tipo) ao usar a função input() em Python para cálculos matemáticos?

R: Para que o programa converta a informação inserida no tipo de dado ideal para a execução do código.

3) O que acontece se tentarmos somar uma variável do tipo str com uma do tipo float?

R: Ocorre TypeError, pois float e string são tipos de dados que não podem ser somados.
